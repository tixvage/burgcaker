# Quick Start 

## Requirements

* C compiler
* Make (will be removed in the feature)

## Compile and Run

```console
$ mkdir build
$ gcc libs/minilua/minilua_exec.c -o build/lua -lm
$ build/lua build-raylib
$ build/lua build-minilua
$ build/lua
```
